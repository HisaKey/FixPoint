フィックスポイント　プログラミング試験について

各設問に対する解答プログラムを作成いたしました。以下各プログラムに関する説明です。なお、プログラミング言語につきましてはPython3を使用しています。

:solver1.py
設問1を解くためのプログラムです。辞書を用いて、あるIPアドレスが初めてタイムアウトしたときの日時とそのIPアドレスを記録し、復旧した際に(IPアドレス,故障開始期間~故障終了期間)の順番に出力するようにしています。

:solver2.py
設問2を解くためのプログラムです。solver1.pyからの変更点として、以前タイムアウトしたIPアドレスが再びタイムアウトしたときに、辞書の同じIPアドレスの部分に値を追加するようにして、復旧した際にそのIPアドレスの要素がN個以上になったときに(IPアドレス,故障開始期間~故障終了期間)の順番に出力するようにしています。なお、Nのデフォルト値は3に設定しています。

:solver3.py
設問3を解くためのプログラムです。solver2.pyからの変更点として、あるIPアドレスに対する日時を値とする辞書(dict_date)と、応答時間を値とする辞書(dict_ping)の二つを設けました。そして、dict_pingに新しいデータの追加し、古いデータの削除を繰り返して、平均応答時間がtを超えたときにその期間を出力するようにしています。なおm,tのデフォルト値はそれぞれm=3,t=5に設定しています。

:solver4.py
設問4を解くためのプログラムです。solver2.pyからの変更点として、IPアドレスをIPとサブネットの二つに分解して、辞書の値にサブネットを使用しています。それ以外の動作はsolver2.pyと変化はありません。

実行方法
使用したいsolver.pyとテストデータを同じディレクトリに置いて
python (使用したいソースコード) < (テストデータ)
と入力することで結果が出力されます。また、ソースコードの番号は設問番号に対応しています。

実行結果について
実行結果についてはresultディレクトリ内部にresult(数字).txtの形で保存しています。また、result(数字).txtの(数字)はテストデータと設問番号に対応しています。
